Installing

The easiest and recommended method of installation is with NuGet by installing the UbietyXMPP package.

Building

To build Ubiety XMPP follow these steps:

  • You will need .NET Framework 4.6 and a C# 6 compatible compiler like Roslyn.

  • Clone the source from http://github.com/ubiety/xmpp

  • Use Visual Studio 2015 or MSBuild 14 to compile the solution.
